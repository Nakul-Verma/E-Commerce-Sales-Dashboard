##E-Commerce Sales Analysis Dashboard (Power BI)

This project presents a comprehensive analysis of e-commerce sales data using Microsoft Power BI, with the objective of deriving actionable business insights and improving decision-making processes. The dashboard is designed to provide a clear and interactive visualization of key performance indicators such as total revenue, profit, order quantity, and customer trends.
The dataset includes transactional level data covering various dimensions such as product categories, customer segments, geographical regions, and time periods. Using Power Query, the raw data was cleaned, transformed, and structured into a well-defined data model following star schema principles. Fact and dimension tables were created to ensure efficient data relationships and optimized performance.

Key features of the dashboard include dynamic filtering, drill-down capabilities, and interactive slicers that allow users to explore sales performance across different regions, categories, and time frames. 
Visualizations such as bar charts, line graphs, pie charts, and KPI cards have been used to highlight trends, seasonal patterns, and top-performing products.

DAX has been extensively utilized to create calculated measures, including year-over-year growth, profit margins, and sales comparisons. These calculations help in identifying business growth patterns, underperforming areas, and potential opportunities for expansion.
The dashboard also provides insights into customer behavior, enabling analysis of purchasing patterns and segmentation. This helps businesses understand their target audience better and make data-driven marketing decisions.

Overall, this project demonstrates strong skills in data cleaning, data modeling, data visualization, and business intelligence. It reflects the ability to convert raw data into meaningful insights using Power BI, making it valuable for stakeholders seeking to optimize sales strategies and improve overall business performance.
